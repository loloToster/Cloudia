# Installation

### 1. Install packages

Default method

```bash
npm run normal-install
```

Raspberry Pi method
```bash
# https://stackoverflow.com/questions/31230606/npm-install-sqlite3-takes-forever
# 1. install sqlite
sudo apt-get install libsqlite3-dev
# 2. run install
npm run rPi-install
```

### 2. Build with

```bash
npm run build
```

### 3. Run with
```
npm start
```
